## Presentation Desk

一个用来培训的高度定制化的 Web 工具。

## 主要功能

- 能按页展示用户准备好的 Deck ，并能在页面上进行必要的教学信息记录或互动。
- 程序为教师自己本地运行即可，几乎共享屏幕给受讲人，没有其他用户访问的需求

### 课程信息准备

从本地指定路径下扫描所有文件夹，一级文件夹可作为课程名称，二级文件夹作为一个主题页面对应的管理文件夹。

二级文件夹内：第一张图片作为该页面的展示图，Readme.md 作为该页面的教学内容，Update.md 记录教师手工编写的信息记录。

### 系统功能

- 选择课程
- 展示大纲，可跳转至对应页面做展示，提供用户操作功能
- 页面展示：页面布局为：
  - 左侧有一个可隐藏的大纲页面 sidebar
  - 主画面左侧 30% 部分，展示图片。图片允许全屏展示或退出全屏。
  - 主画面右侧为 tab 结构，包含以下 tab：
    - 教学内容：展示当前页面的教学内容（读取 readme.md），把 markdown 渲染出来
    - 信息记录：展示当前页面的信息记录（读取 update.md），允许修改（提供保存按钮），提供 preview 按钮展示渲染后的 markdown
    - Web 浏览器：提供一个浏览器，让受讲人可以在不离开这个页面的情况下访问其他必要的网站展示信息。
