# 杭州：人间天堂的现代演绎

> “江南忆，最忆是杭州。” —— 白居易

**杭州**，这座位于中国东南沿海的城市，不仅是浙江省的省会，更是中国七大古都之一。它以风景秀丽的西湖著称，素有“人间天堂”的美誉。

## 1. 必游景点 (Unordered List)

来到杭州，以下景点是绝对不能错过的：

*   **西湖 (West Lake)**：杭州的灵魂。
    *   *苏堤春晓*
    *   *断桥残雪*
    *   *雷峰夕照*
*   **灵隐寺**：千年古刹，佛音袅袅。
*   **西溪湿地**：罕见的城中次生湿地。
*   **宋城**：给我一天，还你千年。

## 4. 出行代码指南 (Code Block)

如果你是一个程序员，来到杭州可能会想写一段代码来记录行程。

```python
# 简单的杭州行程规划器
def plan_trip(budget, days):
    destinations = ["西湖", "灵隐寺", "龙井村"]
    food = ["片儿川", "葱包桧"]
    
    if budget > 2000 and days >= 3:
        print(f"推荐游玩: {', '.join(destinations)}")
        print(f"推荐品尝: {', '.join(food)}")
    else:
        print("建议先去西湖边散散步，免费且惬意！")

plan_trip(2500, 3)
```
