## 涌现“智慧”

GPT3.5 是第一次「涌现能力」（Emergent Ability），本质上是知识语料 + 算力发展突破了某一项阈值。

### GPT4

行业标准，解决了轻易幻觉等问题，并可以通过 Chain of Thought 等提示词工程的方案，让模型强行思考，提高素质。

### OpenAI o1 和 DeepSeek R1 - Reasoning（推理？）

- OpenAI（闭源）通过 Reasoning（预推理）的方式，极大提高了模型的能力边界。
- DeepSeek R1 - 开源 Reasoning 能力几乎同时间探索

### DeepSeek 为什么历史地位高

- 开源 Reasoning 模型，一线水平，打开了开源模型百花齐放的格局
- 极低的训练和运行成本
