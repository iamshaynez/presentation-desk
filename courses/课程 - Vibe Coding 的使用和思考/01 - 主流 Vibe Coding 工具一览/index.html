02. 全球领军者 (Global)

定义行业标准，探索技术边界。

The "Big Three" IDEs

行业标杆
Cursor Anysphere

Vibe Coding 的定义者。它首先让 Tab 键具备了修改多行代码的能力，并首创了 Composer。

  • Composer: 能够创建、编辑、删除多个文件的多模态面板。
  • Shadow Workspace: 隐式运行代码检查错误，不打扰用户。
  • cpp (Codebase++): 极速索引大型代码库。
强力竞品
Windsurf Codeium

Codeium 推出的独立 IDE。主打 "Flows" 概念，强调 AI 对上下文的深层理解，不仅是协作，而是“结对编程”。

  • Cascade Flow: 能够理解最近操作历史的上下文感知流。
  • Deep Context: 相比 Cursor 更擅长理解复杂项目结构。
老牌霸主
Copilot Workspace GitHub / Microsoft

正在从插件向完整工作区进化。依托 GitHub 庞大的数据和生态优势。

  • GitHub Native: 直接从 Issue 或 PR 启动编码任务。
  • Copilot Edits: 类似于 Composer 的多文件编辑体验。

Next-Gen Agents & Web Generators

终端革命
Claude Code Anthropic

生活在 CLI 中的高级代理。不依赖特定编辑器，直接操作文件系统。

系统接管
Operator (Codex) OpenAI

通过 Computer Use API 操作浏览器和开发工具，全能型助手。

Web生成
Replit / Bolt.new StackBlitz 等

浏览器即 IDE。Bolt.new 允许用户通过一句话 Prompt 生成完整的全栈 Web 应用。

开源力量 (Open Source)

OpenCode: Google 开源项目，强调无锁定的 CLI 体验。
PearAI: Cursor 的开源替代品，基于 VS Code，允许用户自带 API Key。

03. 中国本土力量 (Domestic)

结合本土模型优势，从模仿到超越。
本土最佳
Trae 字节跳动 ByteDance

目前国内完成度最高的 AI Native IDE。不仅集成了豆包大模型，还巧妙融合了 Claude 3.5 Sonnet 的能力。

  • 双模设计: 默认 IDE 模式辅助，SOLO 模式全自动接管。
  • 中文优化: 对中文指令和国内开发环境（如 npm 镜像等）有更好支持。
  • 免费策略: 目前阶段提供极其慷慨的免费模型使用额度。
广泛应用
通义灵码 阿里巴巴 Alibaba

国内用户量极大的插件级产品，深度集成于 IDEA 和 VS Code 生态。

  • 企业级: 强调企业代码库的安全与私有化部署。
  • 阿里云集成: 与云端部署、DevOps 流程结合紧密。
  • Qwen 2.5: 底层基于强大的通义千问模型。
早期探索
MarsCode 字节跳动 ByteDance

字节早期的 AI 编程助手，提供云端 IDE 和 IDE 插件。虽然 Trae 是目前的旗舰，但 MarsCode 依然在维护，更侧重云端开发环境。
