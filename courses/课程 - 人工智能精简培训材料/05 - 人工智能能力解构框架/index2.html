如何系统性的设计 AI 实习生的工作？

只有在 知识、信息、介质、逻辑 四个维度均有确信的 Tool 支撑，任务才具备 AI 落地可行性。
知识依赖
任务是依靠通用知识、方法论，还是企业内独有的规则或者隐性知识？
关键工具
  • 知识库 (RAG)
  • 企业文档资料
  • 提示词规则 (Prompt)
信息依赖
依赖外部实时信息、动态数据，还是完全靠大模型内置的训练信息？
关键工具
  • Web Search 联网
  • API 系统对接
  • 人工提供数据
介质依赖
输入输出的数据载体格式？数据库 / Excel / PDF？如何进行格式流转？
关键工具
  • 文件解析器
  • 数据库连接器
  • 格式转换工具
逻辑依赖
需要复杂数据统计计算，还是严格的逻辑规则推理？流程是否非线性？
关键工具
  • 代码解释器
  • 逻辑推理引擎
  • Function Call
核心观点：从小到简单的翻译工作，大到行业核心价值范式，均可用此框架解构。
